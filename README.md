# AutoPartes

En este proyecto se llevó a cabo la toma de una base de datos de un almacén de autopartes de motos de la ciudad de Sincelejo, con el objetivo de que dichos datos sean preprocesados, separados en un conjunto entrenamiento y
prueba, configurados con un algoritmo de clasificación, entrenados en un modelo, arrojen una predicción y finalmente concluyan con una evaluación la cual le permita a la empresa tomar decisiones que ayuden a los procesos de compra y venta de los productos que el almaceno oferte.
